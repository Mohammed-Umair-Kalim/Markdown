# Heading1
## Heading2
### Heading3

Bold:
**Bold Text**

Italic:
*Italic Text*

Strikethrough:
~~Strikethrough~~

Unorderd list:
* item1
+ item2
- item3

Ordered list:
1. first item

Links:
[Youtube](https://www.youtube.com)

Blockquote:
> This is a blockquote

Task list:
- [ ] for an unchecked box
- [x] for a checked box

Escaping character:
\*not italic*\
\**not bold**\

For inline code, use single backticks `code`.
For code blocks, use triple backticks ``` (or indent with 4 spaces).
