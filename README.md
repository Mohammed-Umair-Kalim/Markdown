Code:
- Inline Code:
For example, to print "Hello, World!" in markdown, you can write `print("Hello, World!")` with single backticks \` \`.

- Code Block
If you have a larger code snippet, you can use triple backticks\``` \``` to create a code block:
```python
def greet():
    print("Hello, World!")
    
greet()
```
