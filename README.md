Code:
- Inline Code:
For example, to print "Hello, World!" in Python, you can write `print("Hello, World!")`.

- Code Block
If you have a larger code snippet, you can use triple backticks to create a code block:
```python
def greet():
    print("Hello, World!")
    
greet()
```
